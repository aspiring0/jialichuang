```python
import pandas as pd
from io import StringIO

# 1. 准备数据
data_content = """
日期,产品,销售额,数量,地区
2024-01-01,产品A,15000,100,华东
2024-01-02,产品B,12000,80,华南
2024-01-03,产品A,18000,120,华东
2024-01-04,产品C,9000,60,华北
2024-01-05,产品B,14000,90,华南
2024-01-06,产品A,20000,130,华东
2024-01-07,产品C,11000,70,华北
2024-01-08,产品B,16000,100,华南
2024-01-09,产品A,22000,140,华东
2024-01-10,产品C,13000,80,华北
"""

# 2. 读取数据
# 使用 StringIO 将字符串转换为文件对象供 read_csv 读取
df = pd.read_csv(StringIO(data_content))

# 3. 数据分析
# 需求：计算每个产品的总销售额和平均销量
# 使用 groupby 对 '产品' 分组，然后使用 agg 进行聚合运算
result = df.groupby('产品').agg(
    总销售额=('销售额', 'sum'),
    平均销量=('数量', 'mean')
).reset_index() # reset_index 将分组列变回普通列

# 4. 输出结果
print("各产品销售统计结果：")
print(result)
```

### 代码运行结果预期：

```text
各产品销售统计结果：
     产品   总销售额   平均销量
0  产品A  75000  122.5
1  产品B  42000   90.0
2  产品C  33000   70.0
```

### 代码解析：
1.  **数据读取**：使用 `StringIO` 模拟读取 CSV 文件的过程，`pd.read_csv` 会自动解析表头和数据类型。
2.  **分组聚合**：`df.groupby('产品')` 将数据按产品名称分组。
3.  **多指标计算**：`.agg()` 方法允许一次性计算多个指标：
    *   `总销售额=('销售额', 'sum')`：对“销售额”列求和。
    *   `平均销量=('数量', 'mean')`：对“数量”列求平均值。
4.  **重置索引**：`.reset_index()` 让结果看起来更像一个标准的表格，便于后续导出或查看。
